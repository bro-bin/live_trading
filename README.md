# live_trading
LiveTrading.py에서 시그널에 따라 매매 실행

매매함수는 TradingFunction.py에 모여있음
 1) buy_etf
 2) sell_etf
 3) buy_basket
 4) sell_basket
 5) all_clear 장마감시 일괄매도

 GetBasketQty는 TradingFunction을 위한 모듈

 ClearAll.py는 보유하고 있는 종목 전량매도

---------------------------시그널-----------------------------  
시가 - nav > 2 시그마 : 바스켓 매수(ETF가 과평가 구간)  
시가 - nav < -2 시그마 : ETF 매수  

공매도가 안되기 떄문에 롱 포지션만 가능
